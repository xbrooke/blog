Markdown ​

更新时间

最近更新：2023-9-28

搭建版本：vitepress v1.0.0-rc.20

如果你还不了解 Markdown ，请看我整理的Markdown教程

说明

VitePress 相对于 VuePress 在Markdown中唯一的区别：链接可以直接渲染，不需要语法

基础功能 ​

VitePress 使用 markdown-it 作为解析器，并使用 Shiki 来突出显示语言语法

基本配置

export default defineConfig({
  markdown:{
      //这里填配置项
  },
})
